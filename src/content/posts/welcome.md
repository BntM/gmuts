# Welcome to GMUTS

This is your first blog post. You can edit this content through the Decap CMS admin panel at `/admin`.

## Getting Started

To create new posts:
1. Go to `/admin` and log in with Netlify Identity
2. Click "New Post" in the Posts collection
3. Write your content using the rich text editor
4. Save and publish

## Markdown Support

Your posts support full markdown:

- **Bold text** and *italic text*
- [Links](https://example.com)
- Code blocks
- And more!

Happy writing!
